# Simple WhatsApp Bot

Bot rahisi ya WhatsApp yenye:
- Auto typing indicator
- Auto react kwa messages
- Auto view status
- QR code authentication

## Setup

1. Clone repository hii
2. `npm install`
3. `npm start`
4. Open http://localhost:3000
5. Weka namba na scan QR code

## Hosting

Inaweza ku-host kwenye:
- Heroku
- Bothosting.net
- Railway
